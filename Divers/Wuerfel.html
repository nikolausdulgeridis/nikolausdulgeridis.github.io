   JavaScript Beispiel: Simulation mit zwei Würfeln
   <script>
   var min=1;
   var max=6;
   var runden = 2;
   var versuche = 10000;
   var e= new Array();
   for (var i=0; i<= max*runden; i++) e[i]= 0;
   for (var count=0; count < versuche; count++)
    {
     var punkte= 0;
     for (var i=0; i < runden ; i++) 
       punkte+= Math.floor(Math.random()*(max-min+1)+min);
     e[punkte] = e[punkte] + 1;
    }
   for (var i= min*runden; i<=max*runden ;i++)
     document.write("Punkte "+i+" = "+Math.round(1000*e[i]/versuche)/10+"%<br>");
   </script >
Oft ist es nicht möglich für Ereignisse eine einfache Formel zur Berechnung der Wahrscheinlichkeiten zu finden. In diesen Fällen bietet es sich an, eine Versuchsreihe durchzuführen. Nachteil: die Zuverlässigkeit der Stichprobe ist schwer zu bestimmen.
